# Description of the data

# The dataset is obtained after research done on a tribe called 'Pima'. Data was collected on different parameters from the 768 females of this tribe (as they are more prone to diabetes).

# Description of the parameters in the data:

# 1.Pregnancies: Number of times a woman was pregnant.

# 2.Glucose: Plasma glucose concentration a 2 hours in an oral glucose tolerance test.

# 3.BloodPressure: Diastolic blood pressure (mm Hg).

# 4.SkinThickness: Triceps skin fold thickness (mm).

# 5.Insulin: 2-Hour serum insulin (mu U/ml).

# 6.BMI: Body mass index (weight in kg/ (height in m) ^2).

# 7.DiabetesPedigreeFunction: Diabetes pedigree function 

# 8.Age: Age in years

# 9.Outcome: Whether a woman is diabetic or not (0: Non-Diabetic, 1: Diabetic)
